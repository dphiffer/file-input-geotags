File input geotags

iOS file input geotags minimal test case. Before trying this out, configure your Location Services to enable location access for Camera and Mobile Safari.

  1. Tap the file input below
  2. Choose "Take Photo or Video"
  3. Take a photo, tap "Use Photo"
  4. Observe the EXIF tags don't include GPS info
  5. Take a photo with the Camera app
  6. Tap the file input and choose the second photo
  7. Observe the EXIF tags do include GPS info
